这是一个基于 Tampermonkey 的用户脚本，用于自动化控制 Bilibili 直播观看和 OBS 录制流程。

1. 直播状态自动检测
- 每 10 秒检测一次直播状态
- 开播后自动自动刷新页面进入观看模式
2. 自动网页全屏
- 直播开播后自动触发网页全屏
- 定时检测全屏状态（每 5 秒）
- 自动修复异常退出全屏的问题
- 智能隐藏侧边栏和控制栏
3. OBS 录制自动控制
- 通过 WebSocket 连接 OBS（端口 4455）
- 自动检测录制状态
- 真播停止时自动停止 OBS 录制
- 支持密码认证

同时还要搭配advanced scene switcher开始直播。虽然可以用脚本，但是advanced scene switcher的开播检测要灵敏可靠一些。

适合需要长时间挂机观看直播、自动录制直播内容的用户，特别是需要无人值守的自动化录制场景。
